
## 🌟 Ключові можливості
* Генерація рандомних послідовностей чисел.
* Алгоритми сортування за зростанням (ASC) та спаданням (DESC).
* Пошук дублікатів та аналіз повторюваних елементів.
* Групування членів масиву з динамічним обчисленням суми груп (розмір групи задається користувачем).
* Підтримка роботи з форматом JSON для сучасного обміну даними.

## 🏗 Архітектура проекту
Проект структурований для зручної підтримки та тестування:

| Файл | Опис |
| :--- | :--- |
| header1.h | Оголошення прототипів, опис структури даних та логіки. |
| source.cpp | Реалізація алгоритмів сортування, групування та аналізу. |
| jUnitTest1 | Набір Unit-тестів для перевірки цілісності алгоритмів. |
