Vector Sorting & Analysis
Опис
Програма на C++ для роботи з вектором цілих чисел. Виконує сортування, групування, аналіз повторень та збереження результатів у JSON файл.
Функціонал
 ∙ Генерація випадкового вектора заданого розміру
 ∙ Сортування за зростанням (Bubble Sort ASC)
 ∙ Сортування за спаданням (Bubble Sort DESC)
 ∙ Групування елементів і підрахунок сум груп
 ∙ Обчислення різниці між максимальною та мінімальною сумою груп
 ∙ Підрахунок кількості повторень кожного елемента
 ∙ Збереження результатів у result.json
Структура проєкту
vector/
├── vector.cpp       # Головна програма (main)
├── Header.h         # Функції сортування та аналізу
├── json.hpp         # Бібліотека nlohmann/json
└── result.json      # Результати (генерується автоматично)
Функції (Header.h)
|Функція         |Опис                            |
|----------------|--------------------------------|
|sortASC(v)      |Сортування вектора за зростанням|
|sortDESC(v)     |Сортування вектора за спаданням |
|groupSums(v, n) |Суми груп по n елементів        |
|diffMaxMin(sums)|Різниця між Smax і Smin         |
UnitTest1/
├── UnitTest1.cpp    # Unit тести
└── pch.h
Запуск
 1. Відкрити vector.sln у Visual Studio 2022
 2. Встановити стартовим проєктом vector
 3. Запустити (Ctrl+F5)
 4. Ввести розмір вектора та розмір групи
Unit тести
Проєкт UnitTest1 містить тести для всіх функцій з Header.h.
Запуск тестів: Test → Run All Tests або Ctrl+R, A

|TestSortDESC  |Сортування за спаданням |
|TestGroupSums |Суми груп               |
