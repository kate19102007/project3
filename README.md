Лабораторна робота: Алгоритми обробки числових послідовностей на C++

Що робить програма:
	1. Запитує кількість елементів `size` для формування масиву;
	2. Запитує значення `n` (крок) для розрахунку групових сум;
	3. Генерує послідовність випадкових цілих чисел у діапазоні `[1; 99]`;
	4. Обчислює групові суми: додає числа порціями по `n` елементів і записує результати в `e.txt`;
	5. Виконує аналіз даних:
    	1.  `sortAsc` — сортування чисел за зростанням;
   		2.  `sortDesc` — сортування чисел за спаданням;
    	3.  `sum2` — загальна сума всіх елементів масиву;
    	4.  `sum3` — сума лише унікальних елементів (після видалення дублікатів);
    	5.  `duplicatesCount` — підрахунок кількості повторюваних чисел;
    	6.  `printDifference` — виведення чисел, які мали дублікати;
	6. Експортує звіт** у форматі `data.json`.

Структура файлів:
	1. Main.cpp (або `project1.cpp`) — логіка взаємодії з користувачем та виклик функцій.
	2. functions.h — бібліотека функцій для обробки чисел (сортування, пошук дублікатів, сумування).
	3. e.txt — текстовий вихідний файл із розрахованими сумами по групах.
	4. data.json — структурований звіт із результатами обчислень (S2, S3, V1, V2).

Модульне тестування (Unit Testing):
Проєкт містить два незалежні тестові модулі для перевірки надійності коду:
1.  PositiveTests — перевірка правильної роботи алгоритмів на коректних даних.
2.  NegativeTests — демонстрація поведінки системи при виникненні логічних помилок (навмисно провалені тести).
