# Vector

## 📌 Опис
Цей проєкт реалізує програму на C++, яка працює з векторами: виконує обробку даних, застосовує допоміжні функції та зберігає результати у форматі JSON.

## ✨ Ключові можливості
- Генерація випадкових чисел
- Сортування за зростанням (ASC) та спаданням (DESC)
- Пошук дублікатів та аналіз повторюваних елементів
- Групування елементів із підрахунком суми
- Збереження результатів у форматі JSON

## Архітектура проєкту

Проєкт структурований для зручної підтримки та тестування:

| Файл            | Опис |
|-----------------|------|
| TASK.cpp        | Основна логіка програми |
| func.cpp / func.h | Допоміжні функції |
| vector.1.cpp    | Операції з векторами |
| UnitTest11.cpp  | Модульні тести |
| json.hpp        | Робота з JSON |
| result.json     | Результати виконання |
