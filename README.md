
# 📊 Vector Sorting & Analysis
**C++ застосунок для комплексного аналізу та маніпуляції даними у векторах.**

Ця програма дозволяє генерувати масиви випадкових чисел, проводити їх багаторівневе сортування, групувати елементи для статистичного аналізу та зберігати фінальні результати у сучасному форматі **JSON**.

---

### 🚀 Основний функціонал
Програма пропонує повний цикл обробки даних:
* **Генерація:** Створення вектора заданого розміру з випадковими числами.
* **Сортування:** Реалізація алгоритму **Bubble Sort** у двох напрямках (ASC/DESC).
* **Груповий аналіз:** Розбиття вектора на блоки та обчислення суми кожного блоку.
* **Статистика:** Пошук дельти ($\Delta$) між максимальною та мінімальною сумами груп.
* **Частотний аналіз:** Підрахунок кількості входжень кожного унікального елемента.
* **Експорт:** Автоматичне формування звіту у файл `result.json`.

### 🛠 Технічні деталі (Header.h)

| Функція | Опис дії |
| :--- | :--- |
| `sortASC(v)` | Класичне сортування бульбашкою за **зростанням**. |
| `sortDESC(v)` | Класичне сортування бульбашкою за **спаданням**. |
| `groupSums(v, n)` | Формує новий вектор, де кожен елемент — це сума $n$ елементів оригіналу. |
| `diffMaxMin(sums)` | Обчислює різницю $S_{max} - S_{min}$ для аналізу розкиду даних. |

### 🧪 Модульне тестування (Unit Tests)
Проєкт має високе покриття тестами, що гарантує коректність обчислень:

* ✅ **TestSortASC**: Перевірка правильності порядку від найменшого до найбільшого.
* ✅ **TestSortDESC**: Перевірка реверсивного сортування.
* ✅ **TestGroupSums**: Контроль точності математичного групування.
* ✅ **TestDiffMaxMin**: Валідація логіки пошуку екстремумів.

### 💻 Інструкція з запуску
1.  Відкрийте файл `vector.sln` у **Visual Studio 2022**.
2.  Виберіть проєкт **vector** як стартап-проєкт (Set as Startup Project).
3.  Натисніть `Ctrl + F5` для запуску.
4.  Дотримуйтесь вказівок у консолі: введіть бажаний розмір вектора та параметри групування.
5.  **Для тестування:** Перейдіть у `Test` → `Run All Tests` (`Ctrl+R, A`).
